Hướng dẫn sử dụng file mod

Copy và ghi đè file mod vào thư mục service\contents\Local\GARENA\data

Vui lòng backup lại file gốc nếu có trường hợp xảy ra lỗi.

Lưu ý: trong trường hợp xảy ra lỗi, vui lòng sử dụng file backup của bạn ghi đè lại vào thư mục service\contents\Local\GARENA\data.
Cảnh báo: Chúng tôi sẽ KHÔNG CHỊU TRÁCH NHIỆM vì bất cứ lý do gì liên quan tới tài khoản của bạn trong quá trình sử dụng file mod.
